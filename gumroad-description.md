# Maestro MCP — Premium License

### The most comprehensive MCP server for VS Code.

96 tools across 24 categories. Control every aspect of VS Code through the Model Context Protocol.

---

**Free features (no license needed):**

- File operations — list, find, search, read, create, copy, move, delete
- Text editing — replace lines, insert, undo
- Terminal — execute commands, read output
- Git — status, diff, log, blame, commit, branch, stash
- Editor — focus, workspace info, output channels
- Diagnostics — errors, warnings, editor state
- Debug — sessions, breakpoints, step execution, variables, call stack, evaluate
- Testing — run tests, coverage, results
- And more: selections, clipboard, diff, tasks, notifications, settings, snippets, refactoring, tabs/layout...

**Premium features (this license unlocks):**

- Code completion provider
- Hover information
- Signature help
- Code actions & CodeLens
- Go to definition, find references, call hierarchy
- Document & workspace symbol search
- Document formatting & rename
- Semantic tokens & inlay hints
- Document links, colors, folding ranges

These are VS Code's Language Server Protocol providers — the same data that powers IntelliSense, but accessible to any MCP client (Claude, Cursor, custom agents, etc.)

1. Install the extension from the VS Code Marketplace (free)
2. Free tools work immediately — no license needed
3. Purchase a license here to unlock premium LSP categories
4. Enter your key via Settings > Maestro MCP > License Key
